# NovaForge Game — Roadmap

## Phase 1
- establish module graph
- create compile-safe game scaffolding (`Source/Game/`)
- create world and voxel primitives
- create player state baseline (R.I.G. exo frame)
- create client and server bootstrap

## Phase 2
- inventory and resource primitives
- building placement validation
- salvage loop
- repair and damage hooks
- season configuration
- dev and smoke-test world bootstrap

## Phase 3
- factions
- AI actor baseline
- procedural generation hooks
- HUD schema and visor overlays
- validation suite
- packaging and release pipeline

## Deferred
- low-poly wrapper implementation
- advanced ship systems
- advanced settlements
- nonessential project-local tools
