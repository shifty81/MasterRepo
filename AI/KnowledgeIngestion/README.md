# Knowledge Ingestion

This module handles PDF/code ingestion and embedding for the local knowledge base. It supports:
- Drag-and-drop PDF/code file ingestion
- Parsing and chunking
- Embedding generation (FAISS/Milvus/Weaviate compatible)
- Semantic search integration
- API for Arbiter and Editor
