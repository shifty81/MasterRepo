# Arbiter AI Orchestrator

Arbiter is the central AI orchestrator for the IDE. It provides:
- Conversational tasking and planning
- Reinforcement and feedback loop
- Integration with codegen, asset gen, and PCG workflows
- Access to the knowledge base (PDF/code embeddings)
- API for Editor and Web UI
